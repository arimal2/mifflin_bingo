How it works: There is only one QR code to print or display. Each time someone scans it, the counter advances and they are automatically sent to the next card variation — 1, 2, 3, 4, 5, then back to 1.

Hosting note: The scan counter is stored in this browser's localStorage. Keep this admin page open on your device during the event to track scans. For multi-device events, replace localStorage with a small shared backend.

Adding real images: Replace each null in CARD_IMAGES at the top of the script with the path to your bingo card image (e.g. "./card1.png").
